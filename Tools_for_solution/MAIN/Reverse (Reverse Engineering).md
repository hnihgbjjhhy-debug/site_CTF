
## **Ghidra**
### Описание

**Ghidra** — это бесплатный, open-source фреймворк для реверс-инжиниринга (Software Reverse Engineering, SRE), разработанный и поддерживаемый Национальным агентством безопасности США (NSA). Он представляет собой набор мощных инструментов для анализа скомпилированного кода (бинарных файлов) без исходного кода. Ghidra позволяет выполнять дизассемблирование, декомпиляцию в C-подобный код, отладку, графинг и скриптинг.

Инструмент поддерживает множество процессорных архитектур (x86, ARM, MIPS, PowerPC и т.д.) и форматов исполняемых файлов (PE, ELF, Mach-O). Ghidra работает на Windows, macOS и Linux, имеет GUI и headless-режим для автоматизации. Он идеален для анализа malware, поиска уязвимостей, CTF-задач и исследований. Ghidra часто сравнивают с IDA Pro, но он полностью бесплатный и расширяемый.
### Версии Ghidra (на начало 2026 года)

На январь 2026 года последняя стабильная версия — **12.0** (выпущена 8 декабря 2025 года). Это крупный релиз с новыми функциями, улучшениями и фиксами. Предыдущие значимые версии: 11.x (включая 11.4 в 2025). Разработка активная, обновления выходят регулярно. Требует Java 21+ для работы. Нет платных версий — полностью open-source.
### Основные инструменты

Ghidra — модульный фреймворк с GUI (CodeBrowser) и множеством компонентов:

- **Disassembler** — Дизассемблирование кода в ассемблер.
- **Decompiler** — Декомпиляция в читаемый C-подобный код (с PCode — промежуточным представлением).
- **Debugger** — Отладка (user-mode и kernel-mode, интеграция с GDB, WinDbg, LLDB).
- **Graphing** — Визуализация графов вызовов, контрольного потока (Control Flow Graph), зависимостей.
- **Scripting** — Скрипты на Java или Python (Jython), автоматизация анализа.
- **Extensions/Plugins** — Расширения (GhidraDev для Eclipse/VS Code), поддержка сообщества.
- **Analysis** — Автоматический анализ (авто-определение функций, строк, импортов).
- **Search & Navigation** — Поиск по коду, перекрестные ссылки (xrefs), патчирование.
- **Project Management** — Работа с проектами, совместный анализ (серверный режим).

Дополнительно: поддержка headless-анализа, экспорт/импорт, интеграция с другими инструментами.

## **IDA Free**

### Описание

**IDA Free** — это бесплатная версия популярного интерактивного дизассемблера и декомпилятора от Hex-Rays, предназначенная для **не коммерческого использования** (обучение, хобби, студенты). Она предоставляет базовые возможности IDA Pro для реверс-инжиниринга бинарных файлов: дизассемблирование, ограниченную декомпиляцию, анализ кода.

IDA Free — отличная точка входа для новичков в reverse engineering, CTF и анализ malware. **Ограничения**: нет доступа к IDAPython API и C++ SDK, ограниченная поддержка архитектур (в основном x86/x64), нет продвинутых плагинов, отладчика и облачного декомпилятора. Для полного функционала нужен IDA Pro или IDA Home.

### Версии IDA Free (на начало 2026 года)

IDA Free основана на ядре IDA Pro, но обновляется реже и с ограничениями. На январь 2026 года актуальная версия — **на базе IDA 9.x** (синхронизирована с Pro 9.2/9.3 Beta).

Ранее были версии на базе 8.x/7.x (например, IDA Free 8.3 или 7.7). Обновления бесплатны для non-commercial. Нет отдельного нумерации — следует релизами Pro (9.2 стабильная, 9.3 Beta).

### Основные инструменты

- **Disassembler** → Интерактивный дизассемблер с графами потока управления (CFG) и перекрестными ссылками (xrefs).
- **Decompiler** → Базовая декомпиляция в C-подобный код (локальная, без облачных архитектур).
- **Analysis Tools** → Авто-анализ, поиск строк/импортов, сигнатуры FLIRT.
- **Navigation** → Поиск, патчинг, комментарии, переименование функций.
- **Поддержка** → Форматы PE/ELF/Mach-O, архитектуры x86/x64 (ограниченно другие).
- **Нет**: Отладчика, IDAPython, полного SDK, Lumina, Teams.

Идеально для обучения базовому реверсу.

## **Radare2** 
### Описание

**Radare2** (r2) — это бесплатный, open-source фреймворк для реверс-инжиниринга, анализа бинарных файлов, forensics, debugging и exploiting. Он представляет собой набор утилит командной строки (CLI), которые можно использовать отдельно или вместе. Radare2 поддерживает множество архитектур (x86, ARM, MIPS и т.д.), форматов файлов и платформ (Windows, Linux, macOS, Android).

Инструмент известен своей мощностью, скриптингом (r2pipe, JavaScript, Python), визуальным режимом и плагинами. Radare2 часто используется в CTF, анализе malware, разработке эксплойтов (ROP gadgets, shellcode). Есть официальный GUI — Iaito (Cutter). Radare2 — полностью бесплатный (GPL), активно развивается сообществом.

### Версии Radare2 (на начало 2026 года)

На январь 2026 года последняя стабильная версия — **6.0.8** (codename CleanWheat, выпущена недавно с фиксами и улучшениями). Предыдущие: 6.0.7, 6.0.6. Разработка идёт на GitHub, обновления выходят часто (bug fixes, новые фичи). Рекомендуется устанавливать из git для самой свежей версии.

### Основные инструменты

Radare2 состоит из набора утилит (r2-*):

- **radare2 (r2)** — Основной инструмент: дизассемблер, отладчик, hex-редактор, анализатор.
- **rabin2** — Извлечение информации (символы, импорты, строки, энтропия).
- **rasm2** — Ассемблер/дизассемблер (сборка/разборка инструкций).
- **rahash2** — Хэширование блоков (MD5, SHA и т.д.).
- **radiff2** — Бинарный diff (сравнение файлов, графы).
- **rafind2** — Поиск паттернов/строк в файлах.
- **ragg2** — Генератор shellcode и egg-hunter.
- **rarun2** — Лаунчер для запуска программ с кастомными настройками (env, args).
- **r2pm** — Менеджер пакетов/плагинов для radare2.
- **Дополнительно**: Визуальный режим (V!), графы (VV), эмуляция (esil), отладка (gdb/remote), скриптинг (r2pipe).

Поддержка ROP search, деобфускации, патчинга.
