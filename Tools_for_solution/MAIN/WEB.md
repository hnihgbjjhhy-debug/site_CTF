
## **Burp Suite**

### Описание 

**Burp Suite** — это мощная интегрированная платформа (набор инструментов) для тестирования безопасности веб-приложений. Она разработана компанией **PortSwigger Web Security** (основатель — Dafydd Stuttard) и считается **золотым стандартом** среди пентестеров, bug bounty хантеров и специалистов по кибербезопасности. Burp Suite позволяет перехватывать, анализировать, модифицировать и атаковать HTTP/HTTPS-трафик между браузером и сервером.

Это **man-in-the-middle proxy** (промежуточный прокси), который действует как "посредник" для детального контроля над запросами и ответами. Инструмент помогает находить уязвимости вроде SQL-инъекций, XSS, CSRF, SSRF и многих других.

Burp Suite написан на Java, работает на Windows, macOS и Linux. Он модульный: инструменты взаимодействуют друг с другом (например, запрос из Proxy можно отправить в Repeater или Intruder).

### Версии Burp Suite (на начало 2026 года)

- Community Edition — бесплатная версия для начинающих и обучения. Включает основные ручные инструменты, но без автоматизированного сканера и продвинутых фич.
- Professional — платная (разовая лицензия или подписка, цена ~$449/год на пользователя). Полный набор: автоматизированный сканер, продвинутые атаки, сохранение проектов и т.д. Самая популярная среди профи.
- Enterprise (теперь часто называется Burp Suite DAST) — для крупных компаний: автоматизированное сканирование на масштабе, интеграция в CI/CD, управление командами.

### Основные инструменты (Tools) в Burp Suite

-  **Proxy** — Основной: перехватывает трафик. Настраиваете браузер на прокси Burp (127.0.0.1:8080), устанавливаете CA-сертификат Burp для HTTPS. Можно intercept (останавливать) запросы, модифицировать и forwarding.
-  **Repeater** — Для ручного повторения и модификации запросов. Идеально для тестирования параметров.
-  **Intruder** — Автоматизированные атаки: brute-force, fuzzing, payload injection (например, списки паролей или SQLi payloads).
-  **Scanner** — Автоматический сканер уязвимостей (только в Pro/Enterprise). Находит тысячи уязвимостей с низким уровнем ложных срабатываний.
-  **Spider/Crawler** — Автоматическое сканирование сайта для карты (sitemap).
-  **Sequencer** — Анализ случайности токенов (session ID, CSRF-токены).
-  **Extender** — Магазин расширений (BApps): тысячи плагинов от сообщества (например, Autorize для bypass auth, Logger++ и т.д.).
-  **Collaborator** — Для обнаружения out-of-band уязвимостей (SSRF, blind XSS).
-  **Другие:** Target (карта сайта), Decoder, Comparer, Logger и т.д.

## **sqlmap**

### Описание 

sqlmap — это открытый инструмент для пентестирования (penetration testing), который автоматизирует процесс обнаружения и эксплуатации уязвимостей SQL-инъекций (SQL injection), а также захват базы данных. Он оснащён мощным движком обнаружения, специализированными функциями для пентестеров и различными опциями для fingerprinting баз данных, извлечения данных, доступа к файловой системе и выполнения команд ОС через out-of-band соединения. sqlmap поддерживает множество баз данных, включая MySQL, PostgreSQL, Oracle, SQL Server и другие, и работает на Python 2.7 и 3.x на любой платформе. Это один из самых популярных инструментов в арсенале этичных хакеров, bug bounty hunters и специалистов по информационной безопасности для автоматизированного тестирования веб-приложений на SQLi. Инструмент фокусируется на автоматизации рутинных задач, таких как инъекция payload'ов, обход WAF (Web Application Firewall) с помощью tamper-скриптов и эскалация привилегий в БД.
### Версии sqlmap (на начало 2026 года)

sqlmap развивается активно, и на начало января 2026 года последняя стабильная версия — 1.10. Проект ведётся на GitHub, где регулярно выходят обновления: например, недавние коммиты включают bump года, добавление credits для --tui (Text User Interface), удаление поддержки Python 2.6 и улучшения tamper-скриптов. Нет разделения на "community" и "pro" версии — инструмент полностью бесплатный и open-source под лицензией GNU GPLv2. Для актуальной версии всегда проверяйте репозиторий, так как обновления выходят часто (ежемесячно или чаще).

### Основные инструменты 

sqlmap — это CLI-утилита (командная строка), но с богатым набором модулей и опций. Вот основные:

- **Detection Engine**: Автоматическое сканирование на SQL-инъекции (boolean-based blind, time-based blind, error-based, UNION query, stacked queries и out-of-band).
- **Database Fingerprinting**: Определение типа БД, версии и backend (e.g., MySQL 5.7 on Linux).
- **Data Extraction**: Извлечение баз, таблиц, колонок, дампов данных (e.g., --dump, --dump-all).
- **File System Access**: Чтение/запись файлов на сервере (e.g., --file-read, --file-write).
- **OS Command Execution**: Выполнение команд на сервере (e.g., --os-cmd, --os-shell).
- **Tamper Scripts**: Более 50 скриптов для обхода фильтров/WAF (e.g., space2comment, apostrophemask).
- **API Mode**: sqlmapapi.py для интеграции в другие инструменты (RESTful API).
- **TUI (Text User Interface)**: --tui для графического интерфейса в терминале.
- **Plugins System**: Расширение функционала плагинами.
- **Дополнительные опции**: Поддержка HTTP/1.0 (--http1.0), прокси, Tor, batch-режим, уровень риска/verbosity и многое другое.

Пример базового использования: python sqlmap.py -u "http://example.com/page?id=1" --dbs (сканирует URL на инъекции и перечисляет базы данных).

## **OWASP ZAP**

### Описание

**OWASP ZAP** (Zed Attack Proxy) — это бесплатный, open-source инструмент для тестирования безопасности веб-приложений (DAST — Dynamic Application Security Testing). Он разработан под эгидой OWASP (Open Worldwide Application Security Project) и активно поддерживается международным сообществом волонтёров. ZAP работает как **man-in-the-middle proxy**: перехватывает трафик между браузером и сервером, позволяет анализировать, модифицировать запросы/ответы и автоматически находить уязвимости (SQLi, XSS, CSRF, SSRF и т.д.).

Инструмент подходит как для новичков (пассивное сканирование, HUD-интерфейс), так и для профессионалов (активные атаки, скриптинг). ZAP часто сравнивают с Burp Suite, но он полностью бесплатный, легче и имеет мощный Marketplace с аддонами. Поддерживает автоматизацию, интеграцию в CI/CD, API-тестирование и работу в headless-режиме (без GUI).

### Версии owasp zap (на начало 2026 года)

На январь 2026 года актуальная стабильная версия — **2.17.0** (выпущена в конце 2025 года). Это полная версия с улучшениями в сканировании, поддержкой Java 17+, новыми аддонами и фиксами.

Еженедельные сборки (Weekly) выходят регулярно (например, D-2025-12-29). Предыдущие значимые релизы: 2.16.0 (начало 2025, с новым client-side spider и улучшениями аутентификации). ZAP не имеет платных версий — всё бесплатно и open-source. Требует Java 17+ (для Windows/Linux; macOS включает Java).

### Основные инструменты

ZAP — модульный инструмент с GUI (десктоп), но поддерживает daemon-режим и API. Ключевые компоненты:

- **Proxy** — Перехват трафика (intercept), поддержка HTTPS (с установкой CA-сертификата).
- **Spider/Crawler** — Автоматическое обнаружение структуры сайта (традиционный + AJAX/Client-side spider для JS-heavy приложений).
- **Passive Scan** — Пассивное сканирование (анализ трафика без атак, находит уязвимости "на лету").
- **Active Scan** — Активное сканирование (инъекции, fuzzing, известные атаки).
- **Fuzzer** — Тестирование на ввод неожиданных данных (payloads из списков или кастомные).
- **HUD (Heads-Up Display)** — Интерфейс прямо в браузере для ручного тестирования (идеально для новичков).
- **Scripts** — Скриптинг (JavaScript, Zest) для кастомных атак.
- **Add-ons Marketplace** — Тысячи расширений: импорт Postman/GraphQL, автоматизация, отчёты и т.д.
- **Automation Framework** — Для CI/CD (YAML-конфиги сканов).
- **Reports & Alerts** — Генерация отчётов (HTML, XML), интеграция с OWASP Top 10.

Дополнительно: поддержка API (REST), daemon-режим, Docker-образы.
