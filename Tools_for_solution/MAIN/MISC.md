## Python
### Описание

**Python** — это высокоуровневый интерпретируемый язык программирования общего назначения, известный своей простотой, читаемостью и универсальностью. Созданный Гвидо ван Россумом в 1991 году, Python следует философии "The Zen of Python" (PEP 20): "Простое лучше сложного", "Читаемость имеет значение". Язык поддерживает несколько парадигм программирования: процедурную, объектно-ориентированную, функциональную и аспектно-ориентированную. Python имеет огромную экосистему библиотек (через менеджер пакетов pip) и применяется практически во всех областях IT: веб-разработка (Django, Flask), анализ данных (Pandas, NumPy), машинное обучение (TensorFlow, PyTorch), автоматизация, скриптинг, DevOps, кибербезопасность, CTF (особенно PPC — Programming/Puzzle Challenges), научные вычисления и многое другое. Благодаря динамической типизации и автоматическому управлению памятью код на Python пишется быстро и с минимальным количеством строк.

### Основные инструменты для решения MISC задач

Misc (Miscellaneous) — одна из самых разнообразных категорий в CTF: сюда входят задачи на стеганографию, анализ файлов, кодировки, сжатие, сетевые протоколы, скриптинг, головоломки, OSINT-подобные вещи и многое другое. Python идеально подходит для них благодаря встроенным модулям и мощным библиотекам.

Вот список **основных и наиболее часто используемых** Python-инструментов для Misc (актуально на 2026 год):

#### Встроенные модули (не требуют установки)

- **base64, binascii, codecs** — работа с кодировками (base64, base32, hex, rot13, urlencode и т.д.).
- **zlib, gzip, bz2, lzma** — распаковка сжатых данных (часто встречается в стего или файлах).
- **hashlib** — вычисление хэшей (MD5, SHA1/256, CRC32 и т.д.).
- **struct** — распаковка бинарных структур.
- **os, subprocess** — взаимодействие с ОС, запуск внешних команд (steghide, exiftool и т.д.).
- **re** — регулярные выражения (парсинг логов, поиск паттернов).
- **socket, telnetlib, http.client** — простые сетевые взаимодействия (misc с сокетами).
- **wave, audioop** — базовая работа с аудио (stego в WAV).
- **json, pickle** — парсинг/дамп данных.

#### Сторонние библиотеки (устанавливаются через pip)

- **Pillow (PIL)** — must-have для стеганографии в изображениях: LSB, анализ метаданных, манипуляции с PNG/JPG/BMP.
- **opencv-python (cv2)** — продвинутый анализ изображений и видео (стего, QR-коды, визуальные пазлы).
- **numpy** и **scipy** — обработка сигналов, аудио-стего, FFT для поиска скрытых данных в спектрах.
- **pydub** — удобная работа с аудио (конвертация, стего в MP3/WAV).
- **qrcode** — генерация/чтение QR-кодов (часто нужно для задач с повреждёнными QR).
- **Ciphey** — автоматический декриптор: сам определяет и расшифровывает кодировки, шифры, хэши. Очень полезен для быстрых misc с цепочками преобразований.
- **steghide** / **outguess** / **binwalk** — CLI-инструменты, но их можно вызывать через subprocess из Python.
- **exiftool** (через subprocess) или **piexif** — чтение/изменение EXIF-метаданных.
- **requests** — скачивание файлов, взаимодействие с веб-сервисами в misc-задачах.
- **beautifulsoup4** — парсинг HTML в задачах с веб-скрапингом или скрытыми данными на страницах.

#### Специализированные для определённых подтипов Misc

- **pwntools** — иногда используется для простых сетевых misc или взаимодействия с бинарными протоколами.
- **construct** — мощная библиотека для парсинга и создания бинарных форматов (альтернатива struct).
- **bitstring** или **bitarray** — удобная работа с битами (LSB-стего).

#### Пример установки основных библиотек

text

```
pip install pillow opencv-python numpy scipy pydub qrcode ciphey requests beautifulsoup4 construct
```

#### Рекомендация для CTF

В большинстве Misc-задач достаточно:

- Pillow + base64/zlib + Ciphey + requests Это комбо решает 80% задач быстро.

Если задача специфическая (например, PDF-стего, ZIP-анализ, Morse), часто помогают встроенные модули + subprocess для вызова внешних утилит (7z, ffmpeg, tesseract для OCR).

## Bash
### Описание

**Bash** (Bourne Again SHell) — это командная оболочка (shell) и язык сценариев для Unix-подобных систем (Linux, macOS, BSD). Это свободная реализация Bourne shell (sh) с добавлением полезных функций из KornShell (ksh) и C Shell (csh). Bash является **стандартной оболочкой** в большинстве дистрибутивов Linux и macOS. Он поддерживает интерактивный режим (командная строка), скриптинг, историю команд, автодополнение (Tab), перенаправление ввода/вывода, пайпы, переменные, циклы, условия и многое другое. Bash идеален для автоматизации задач, администрирования и решения Misc-задач в CTF (где часто нужно манипулировать файлами, искать строки, обрабатывать текст).

### Версии Bash (на начало 2026 года)

На январь 2026 года актуальная стабильная версия — **5.3** (выпущена в 2025 году). Это последняя major-версия с улучшениями в обработке команд, совместимости и новых фичах (например, улучшенная подстановка команд в текущем контексте). Большинство Linux-дистрибутивов используют версии 5.1–5.2, но обновления до 5.3 доступны. Проект поддерживается Chet Ramey в рамках GNU.

### Основные инструменты для решения Misc задач

В CTF Misc-задачах Bash — основной инструмент для обработки файлов, поиска флагов и автоматизации. Вот ключевые команды и приёмы:

- **Поиск и grep**:
    - grep -r "flag{" . — рекурсивный поиск строки в файлах.
    - grep -a "text" binary_file — поиск в бинарниках.
    - find / -name "*.txt" 2>/dev/null — поиск файлов по имени.
- **Обработка текста**:
    - strings file.bin — извлечение строк из бинарника.
    - cat file | sort | uniq — сортировка и удаление дубликатов.
    - xxd file или hexdump -C file — hex-дамп.
    - base64 -d file — декодирование base64.
- **Файлы и архивы**:
    - file file — определение типа файла.
    - tar -xf archive.tar, unzip file.zip — распаковка.
    - dd if=file bs=1 skip=N count=M — извлечение байтов.
- **Пайпы и перенаправление**:
    - command1 | command2 — цепочки (e.g., cat file | grep flag).
    - command > file / >> file — вывод в файл.
- **Скриптинг в one-liner**:
    - for f in *.txt; do cat "$f"; done — обработка всех файлов.
    - while read line; do echo "$line"; done < file — построчная обработка.
- **Другие полезные**:
    - nc -lvp port — слушатель для reverse shell.
    - curl/wget — скачивание файлов.
    - env | grep FLAG — поиск переменных окружения.
