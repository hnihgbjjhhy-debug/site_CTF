
## Foundry
### Описание

**Foundry** — это быстрый, портативный и модульный toolkit для разработки приложений на Ethereum, написанный на Rust. Он предназначен для создания, тестирования, деплоя и взаимодействия со смарт-контрактами на Solidity. Foundry популярен среди разработчиков блокчейн-приложений (включая CTF в категории Blockchain/Web3), bug bounty и аудита контрактов благодаря скорости, простоте и интеграции. Это альтернатива Hardhat, Truffle или Brownie, но быстрее и без зависимости от Node.js. Foundry поддерживает EVM-совместимые цепи (Polygon, Arbitrum, Optimism и т.д.), fuzzing, forking mainnet и многое другое.

### Версии Foundry (на начало 2026 года)

Проект активно развивается с nightly-билдами (ежедневными). На январь 2026 года актуальны **nightly-релизы** (например, Nightly от 2026-01-02 и позже). Стабильные версии выходят реже (последние крупные — 1.x в предыдущие годы), но рекомендуется использовать nightly через foundryup для свежих фич. Нет жёсткого разделения на "community/pro" — весь toolkit open-source и бесплатный.

### Основные инструменты

Foundry состоит из четырёх ключевых CLI-утилит:

- **Forge**: Фреймворк для сборки, тестирования (включая fuzzing), отладки и деплоя смарт-контрактов. Поддержка скриптов на Solidity, интеграция с git submodules для зависимостей (например, OpenZeppelin).
- **Cast**: "Швейцарский нож" для взаимодействия с EVM: отправка транзакций, вызовы контрактов, чтение данных цепи, конвертация значений и т.д.
- **Anvil**: Локальный Ethereum-нод (аналог Ganache/Hardhat Network). Поддержка forking mainnet/testnet для тестирования на реальных данных.
- **Chisel**: Быстрый REPL (Read-Eval-Print Loop) для Solidity — интерактивная оболочка для тестирования кода на лету.

Все инструменты интегрированы и работают из коробки.

### Ссылка для скачивания

Официальный сайт и установка: **[https://getfoundry.sh/](https://getfoundry.sh/?referrer=grok.com)** или репозиторий **[https://github.com/foundry-rs/foundry](https://github.com/foundry-rs/foundry?referrer=grok.com)**

Рекомендуемый способ установки (через foundryup):

text

```

```

Это установит forge, cast, anvil и chisel (latest nightly). Для обновлений просто запустите foundryup.

## Etherscan
### Описание

**Etherscan** — это ведущий блокчейн-эксплорер (block explorer) и аналитическая платформа для сети Ethereum и совместимых с EVM цепочек. Он позволяет пользователям просматривать и искать транзакции, адреса, токены (ERC-20, ERC-721/NFT), смарт-контракты, блоки, газовые цены и другую активность в блокчейне Ethereum в реальном времени. Etherscan часто называют "Google для Ethereum", поскольку он предоставляет прозрачный и удобный интерфейс для анализа децентрализованной сети. Платформа поддерживает верификацию смарт-контрактов, отслеживание балансов, историю транзакций, газ-трекер и аналитику. Это незаменимый инструмент для разработчиков, трейдеров, инвесторов и энтузиастов Web3/Blockchain (включая CTF и bug bounty в категории Blockchain/Web3).

### Версии Etherscan (на начало 2026 года)

Etherscan — это веб-сервис, который постоянно обновляется без строгих версионированных релизов платформы. На начало 2026 года актуальна **API V2** (выпущена в бета в 2024 году, полная версия в 2025), которая унифицирует доступ к данным более 50 EVM-совместимых цепочек под одним API-ключом. Основные обновления включают мультичейн-поддержку (Blockscan для 25+ цепочек), улучшенный газ-трекер и интеграции для L2-решений (например, Base, OP Mainnet). Нет десктопных или мобильных "версий" — только веб и API.

### Основные инструменты

- **Поиск и просмотр**: Транзакции, адреса (баланс, токены, NFT), блоки, смарт-контракты (верификация кода).
- **Газ-трекер**: Реальное время газовых цен для оптимизации транзакций.
- **Аналитика**: Статистика сети, топ-токены, чарты, токен-трекер.
- **API**: Бесплатный и Pro-уровни для разработчиков (модули: account, contract, transaction, logs, stats и т.д.).
- **Дополнительно**: Верификация контрактов, приватные теги адресов, экспорт данных, мультичейн-эксплореры (для тестнетов как Sepolia, других цепочек как BSC, Polygon).
- **Blockscan**: Единый поиск по 25+ цепочкам.

### Ссылка для скачивания

Etherscan — это веб-сервис, его нельзя скачать как приложение. Доступ через браузер: **[https://etherscan.io/](https://etherscan.io/?referrer=grok.com)**

Для разработчиков (API-клиенты/обёртки):

- Python: pip install py-etherscan-api или аналогичные библиотеки.
- Node.js: npm install etherscan-api
- Официальная документация API: **[https://docs.etherscan.io/](https://docs.etherscan.io/?referrer=grok.com)** (регистрация для API-ключа бесплатная).
