
## Python
### Описание

**Python** — это высокоуровневый интерпретируемый язык программирования общего назначения, известный своей простотой, читаемостью и универсальностью. Созданный Гвидо ван Россумом в 1991 году, Python следует философии "The Zen of Python" (PEP 20): "Простое лучше сложного", "Читаемость имеет значение". Язык поддерживает несколько парадигм программирования: процедурную, объектно-ориентированную, функциональную и аспектно-ориентированную. Python имеет огромную экосистему библиотек (через менеджер пакетов pip) и применяется практически во всех областях IT: веб-разработка (Django, Flask), анализ данных (Pandas, NumPy), машинное обучение (TensorFlow, PyTorch), автоматизация, скриптинг, DevOps, кибербезопасность, CTF (особенно PPC — Programming/Puzzle Challenges), научные вычисления и многое другое. Благодаря динамической типизации и автоматическому управлению памятью код на Python пишется быстро и с минимальным количеством строк.

### Основные инструменты для решения PPC задач

В CTF (особенно в категории PPC — Professional Programming Challenges или Puzzle/Programming) Python является одним из самых популярных языков благодаря встроенным возможностям и богатой экосистеме. Ключевые инструменты и библиотеки:

- **Встроенные модули**:
    - socket, http.client — сетевые задачи.
    - hashlib, hmac — криптография (MD5, SHA, HMAC).
    - base64, binascii — кодировки.
    - struct — работа с бинарными данными.
    - itertools, collections — комбинаторика и алгоритмы.
    - re — регулярные выражения.
    - math, decimal — математические расчёты.
- **Сторонние библиотеки** (устанавливаются через pip):
    - **pwntools** — must-have для pwn и эксплойтов (ассемблер, ROP, взаимодействие с процессами).
    - **z3** (от Microsoft) — SMT-солвер для решения систем уравнений и логических задач (крипто, реверс).
    - **sympy** — символьные вычисления, решение уравнений.
    - **pycryptodome** (Crypto) — полноценная криптография (AES, RSA, ECC и т.д.).
    - **requests** — удобная работа с HTTP.
    - **gmpy2** — быстрые большие числа и факторизация.
    - **angr** — символьное выполнение для реверса.
    - **ropper/ROPgadget** — поиск ROP-гаджетов.

Python идеален для быстрого прототипирования решений в CTF: написать скрипт для brute-force, взаимодействия с сокетом или решения крипто-задачи можно за минуты.

## Z3 solver
### Описание

**Z3** (Z3 Theorem Prover) — это высокопроизводительный **SMT-солвер** (Satisfiability Modulo Theories), разработанный Microsoft Research. Он предназначен для проверки удовлетворимости логических формул в различных теориях (арифметика, бит-векторы, массивы, строки, floating-point и т.д.). Z3 широко используется в задачах верификации ПО, анализа безопасности, CTF (категории Crypto, PPC, Reverse — для решения систем уравнений, взлома шифров, анализа ограничений), символическом выполнении, fuzzing и исследованиях. Солвер поддерживает формат SMT-LIB2, имеет API для Python, C/C++, .NET, Java, OCaml и других языков. Z3 — open-source под MIT-лицензией, активно развивается и считается одним из лучших в своём классе.

### Версии Z3 solver (на начало 2026 года)

На январь 2026 года последняя версия — **4.15.4.0** (или близкая к ней, как 4.15.x). Это видно из пакета Python-биндингов z3-solver на PyPI. Основная ветка разработки — 4.x (с 2015 года), с регулярными обновлениями (коммиты до декабря 2025). Предыдущие версии: 4.13.x, 4.12.x и т.д. Нет разделения на "community/pro" — весь функционал бесплатный и открытый.

### Основные инструменты

Z3 предоставляет **Solver** для проверки satisfiability (sat/unsat/unknown), модели (примеры значений переменных), оптимизации и так далее. Ключевые компоненты:

- **Теории и типы**: Int, Real, Bool, BitVec, Array, FloatingPoint, String, Datatype и т.д.
- **API (на примере Python)**:
    - Solver(): Создание солвера, add() для добавления ограничений, check() для проверки (sat/unsat).
    - model(): Получение модели (значений переменных).
    - Optimize(): Для задач оптимизации (maximize/minimize).
    - Тактики и стратегии: Для кастомизации решения сложных задач.
- **CLI**: Запуск z3 file.smt2 для файлов в формате SMT-LIB.
- **Интеграции**: Биндинги для многих языков; в CTF часто используется Python-API для скриптов (например, решение систем уравнений в реверсе или крипто).
- **Дополнительно**: Поддержка unsat-core, proof, interpolation, parallelism.

Пример использования в Python:

Python

```
from z3 import *
x = Int('x')
s = Solver()
s.add(x > 0, x < 10, x % 2 == 0)
if s.check() == sat:
    print(s.model())  # [x = 2] или подобное
```
